🛡️ Matrice des rôles & permissions
Module / Rubrique 👑 Super Admin 🔑 Administrateur 👤 Agent ✍️ Ambassadeur 🏛️ Insp. MAE
📖 Description des rôles
👑 Super Administrateur
Accès complet à toutes les rubriques. Peut créer, modifier et supprimer tous les comptes. Gère les rôles et paramètres système.
🔑 Administrateur
Gère les dossiers, contacts, documents et rendez-vous. Peut créer des comptes agents. N'a pas accès aux paramètres système ni à la gestion des rôles.
👤 Agent
Génère des documents, gère les rendez-vous et dossiers qui lui sont assignés. Accès lecture aux contacts. Pas d'accès à l'administration.
✍️ Ambassadeur
Accès au tableau de bord et à la file de signature uniquement. Peut approuver ou rejeter les documents soumis pour signature officielle.
🏛️ Inspecteur MAE
Accès exclusif à la Vue Ministère. Tableau de bord centralisé de toutes les ambassades algériennes — statistiques globales, dossiers, alertes, signatures. Réservé au personnel du Ministère des Affaires Étrangères.

🌐 Étape 1 — Créer un compte EmailJS gratuit

EmailJS permet d'envoyer de vrais emails Gmail directement depuis le navigateur, sans serveur.

Plan gratuit : 200 emails/mois — suffisant pour commencer.
Pour plus de volume, le plan payant commence à $9/mois (1 500 emails/mois).
  1. Va sur www.emailjs.com ↗
  2. Clique sur "Sign Up Free"
  3. Crée ton compte avec ton email Gmail de l'ambassade
  4. Confirme ton adresse email

📬 Étape 2 — Connecter ton compte Gmail

  1. Connecte-toi à ton dashboard EmailJS
  2. Dans le menu gauche, clique sur "Email Services"
  3. Clique sur "Add New Service"
  4. Choisis "Gmail"
  5. Clique sur "Connect Account" et autorise avec ton compte Gmail de l'ambassade
  6. Donne un nom au service, par exemple : ambassade_gmail
  7. Clique sur "Create Service"
  8. Note le Service ID (ex: service_xxxxxxx)

📝 Étape 3 — Créer le template d'email

  1. Dans EmailJS, clique sur "Email Templates" dans le menu
  2. Clique sur "Create New Template"
  3. Configure le template comme suit :
To : {{to_email}}
From name : {{from_name}}
Subject : {{subject}}
Content :
{{message}}

---
Ce message vous a été envoyé par l'Ambassade d'Algérie.
  • 4. Clique sur "Save"
  • 5. Note le Template ID (ex: template_xxxxxxx)

    • 🔑 Étape 4 — Récupérer ta clé publique

    1. Dans EmailJS, clique sur ton profil en haut à droite
    2. Va dans "Account" → "General"
    3. Copie ta Public Key (ex: user_xxxxxxxxxxxxxxxxx)
